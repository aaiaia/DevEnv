# Installation

## Java
### ubuntu
1. `apt-get update`
2. `apt-get install openjdk-13-jdk`
3. add shell variable `JAVA_HOME="/usr/lib/jvm/java-13-openjdk-amd64"` to `/etc/environment` or user shell rc file like `~/.zshrc`, `~/.bashrc`
